Estructura de Texto

La etiqueta <b> </b> se usa para agregar estilo visul en bold

La etiqueta <strong> </strong> le dá enfasis en el navegador y el buscador. Se debe moderar su uso

La etiqueta <span> </span> no tiene un efecto visual por si solo, solo con CSS se modifica el estilo

La etiqueta <i> </i> pone en italica el contenido

La etiqueta <blockquote> </blockquote> agrega una cita

La etiqueta <cite> </cite> se usa para agregar el autor de la cita

La etiqueta <hr> </hr>agrega una linea separadora
